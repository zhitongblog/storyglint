快速上手指南

1 首次使用：配置 Gemini API Key

1

获取 API Key

访问 Google AI Studio 登录你的 Google 账号，点击「Create API Key」生成一个免费的 API Key。

2

打开全局设置

启动 StoryGlint 后，点击左上角的 全局设置 按钮进入设置页面。

3

填入 API Key

在「Gemini API Key」输入框中粘贴你的 Key，点击「测试配额」确认 Key 有效。如果在中国大陆使用，需要配置代理地址。

4

选择模型

点击「查找可用模型」，系统会自动检测你的 Key 可以使用的模型。推荐选择 gemini-2.0-flash 或 gemini-1.5-pro。

⚠️ 注意事项

Gemini API 免费版有每日调用限制。如果遇到「配额不足」错误，可以等待24小时后重试，或更换新的 API Key。

2 创作要点与技巧

💡 灵感输入

创建新书时，在「灵感」栏中详细描述你的故事构思，包括主角设定、核心冲突、世界观等。描述越详细，AI 生成的大纲质量越高。

📖 大纲规划

建议先生成卷大纲，再逐卷生成章节细纲。每卷完成后再进行下一卷，保持故事连贯性。

✏️ 章节创作

AI 生成的章节内容是初稿，建议在此基础上进行修改润色，加入你的个人风格和细节描写。

👥 角色管理

及时在角色管理中添加和更新角色信息，系统会在后续创作中自动参考这些设定，保持角色一致性。

💾 定期备份

开启云端同步功能，登录账号后作品会自动备份到云端。也可以使用导出功能将作品保存到本地。

🔄 善用重新生成

对生成结果不满意？可以点击重新生成按钮获取新的内容。每次生成都会有所不同，多试几次往往能得到满意的结果。
